## Getting Started <a name = "getting_started"></a>

### Prerequisites

-   Mengunnakan PostgreSQL sebagai database
-   Setup env variable

```
# Golang

PORT=

DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_NAME=
DB_SSLMODE=
```

```
# Laravel

DB_CONNECTION=psql
DB_HOST=
DB_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
```
